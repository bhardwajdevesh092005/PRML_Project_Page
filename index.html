Speaker Recognition using Machine Learning

Recognizing speakers based on their voice signatures

Abstract

This project presents a real-time speaker recognition system developed using machine learning techniques. The system extracts audio features such as MFCC, delta MFCC, and delta-delta MFCC, and classifies speakers using various models, including a Convolutional Neural Network. The dataset used for training and testing consists of audio samples from multiple speakers. The system achieves high accuracy in recognizing speakers, demonstrating the effectiveness of machine learning in audio processing tasks. The project aims to provide a robust solution for applications in security, voice biometrics, and human-computer interaction.

Techniques Used

Features used:

  1. MFCC
  2. Delta MFCC
  3. Delta-Delta MFCC

Models used

    Classical ML:
    • K Nearest Neighbors
    • XG Boost
    • Random Forest
    • K Means Clustering
  1. Deep Learning:
    CNN:

    2. CNN was applied on multi-dimensional array of MFCCs delta-MFCCs and delta2-MFCCs to obtain a high accuracy on the test Data.

    Embeding Models:

    Embedings are generated using a pretrained model and sotred in a vector database, then when a test audio arrives the embedings are generated and compared with the stored databse and the best match speaker is identified.

Dataset

We use publicly available datasets from Kaggle for training and testing our models. The dataset contains audio samples from 50 speakers, which are pre-processed to extract features like MFCCs. The dataset is split into training and testing sets to evaluate the performance of our models.

Results

  • CNN Model: ~92% accuracy
    • KNN: ~27% accuracy
    • XGBoost: ~69.5% accuracy
    • Random Forest: ~57% accuracy
    • Support Vector Machine: ~48% accuracy
